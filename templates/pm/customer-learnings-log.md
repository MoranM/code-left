## How to add a row

1. **Learning** — Specific, falsifiable where possible.
2. **Source** — Enough detail to find the original (person, study, ticket ID range, dashboard name and date range).
3. **Why it matters** — Which decisions this should influence.
4. **Confidence** — High only with strong evidence; use low for anecdotes.
5. **Related bets** — Link to bet briefs or issue IDs when relevant.
